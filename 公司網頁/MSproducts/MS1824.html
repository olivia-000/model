MS1824

HDMI Video Processor

General Description

MS2130S is a USB 3.0 HD video and audio capture chip, which integrates USB 3.0 device controller, data transceiver module, video and audio processing module. MS2130S can transmit the HD input audio and video signals to PC, smartphone and tablet for preview or acquisition through USB 3.0 interface. MS2130S output supports YUV422, MJPEG, RGB24 and NV12 modes, compatible with Windows, Android and macOS systems.

HD RX Audio

◆ Supports deep color 24/30/36bit mode
◆ Supports CEA-861-E, CEA-861-F video format
◆ Max input resolution:
4096 x 2160@60Hz YUV420
4096 x 2160@30Hz YUV444
◆ Supports HDCP1.4
 ◆ USB audio default 48kHz/24bit stereo output
